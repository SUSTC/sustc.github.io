# 南科大 Gitbook

项目托管在 [SUSTC/sustc.github.io](https://github.com/sustc/sustc.github.io)

## 如何贡献
1. Pull request
    1. Fork 后，进行修改。
    2. 发起 Merge 请求。

2. 代理提交
    1. 进入[GitBook编者 QQ群](https://jq.qq.com/?_wv=1027&k=5D8EgDF): 905618908
    2. 提交规范的Word，Markdown 文件给管理员。
    3. 管理员整理上传。

## 编写规范

* 使用 Markdown 和 raw HTML / JavaScript 语言
